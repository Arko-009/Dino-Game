# 🦖 Dino Game (JavaScript)

A simple and fun browser-based Dino Game built using **HTML, CSS, and JavaScript**.
Inspired by the classic offline Chrome dinosaur game, this project demonstrates core front-end concepts like DOM manipulation, animations, and event handling.

---

## 🚀 Live Demo

👉 https://arko-009.github.io/Dino-Game/

## ✨ Features

* 🕹️ Smooth gameplay with keyboard controls
* ⚡ Real-time obstacle movement
* 🎯 Score tracking system
* 🔁 Game restart functionality
* 🎨 Clean UI with responsive layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
